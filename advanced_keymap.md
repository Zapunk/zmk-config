# Lily58 ZMK Configuration

Персонализированная раскладка для программирования с использованием Home Row Mods и слоев навигации.

## Особенности
- **Home Row Mods**: Модификаторы (Shift, Ctrl, Alt, Gui) расположены на основной строке клавиш (A, S, D, F / J, K, L, ;).
- **Combos**: 
  - `J + K` = Escape
  - `D + F` = Tab
- **Tri-Layer**: Одновременное зажатие слоев Lower (1) и Raise (2) активирует слой Adjust (3).

## Слои

### 0: Base (Основной)
Стандартная QWERTY раскладка с Home Row Mods:
- `F / J` -> LGUI / RGUI
- `D / K` -> LSHIFT / RSHIFT
- `S / L` -> LCTRL / RCTRL
- `A / ;` -> LALT / RALT

### 1: Symbols (Символы)
Оптимизирован для кодинга. Скобки `{ [ ] }` и знаки `! @ # $ %` вынесены ближе к центру.

### 2: Nav (Навигация)
Стрелки и управление курсором на правой руке (H, J, K, L). Удобно для перемещения по коду без отрыва рук.

### 3: Adjust (Настройки)
Управление Bluetooth (очистка профилей, выбор устройств) и переключение вывода (USB/BLE).
