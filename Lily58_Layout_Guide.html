Макросы

  • ⌘C — Copy
  • ⌘V — Paste
  • Ctrl+Shift+C — Terminal Copy
  • Ctrl+Shift+V — Terminal Paste
  • ⌘A — Select All
  • ⌥← / ⌥→ — переход по словам

Bluetooth / Adjust

  • BT0–BT4 — переключение между устройствами.
  • BT CLR — очистить все Bluetooth-пары.
  • OUT TOG — переключить USB ↔ Bluetooth.
  • RESET — перезагрузка контроллера.
  • BOOT — вход в bootloader для прошивки.

Переключение языка

Для MacOS использовать горячую клавишу MacLang = CTRL + SPACE.

Для Linux использовать горячую клавишу LinxLang = ALT + SHIFT.

Горячие клавиши

  • ⌘C: удерживать K + C
  • ⌘V: удерживать K + V
  • ⌘⇧P: удерживать K + J + P
  • ⌘F: удерживать K + F
  • ⌘/: удерживать K + /
